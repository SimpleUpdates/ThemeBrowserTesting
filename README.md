# ThemeBrowserTesting

Installation:

Installation:

```bash
composer install
chmod +x run.sh
```

Usage:

```bash
./run.sh /path/to/theme
```

Rebuild the app:

```bash
docker-compose build
```

Run the app and jump into terminal:

```bash
docker-compose run --entrypoint bash app
```
